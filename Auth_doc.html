Auth: сервис аутентификации¶

1. Создание нового пользователя¶

Запрос: /auth/new
Тело: 

{
    login:"drankov",
    password:"12345678" 
}

Ответ: 
{
    error_code: <error_code> 
}

Коды результата (error_code):

  • 0 — успешное создание
  • 1 — пользователь создан небыл

2. Получение токена¶

Запрос: /auth/auth
Тело: 

{
    login:"drankov",
    password:"12345678" 
}

Ответ: 
{
    error_code: 0,
    token:"12312312j3jijri1ojrIJIO+124+-7JROIRJ" 
}

или
{
     error_code: 1,
     error_msg: <error_msg>
}

3. Проверка токена на валидность¶

Запрос : /auth/check
Тело запроса: 

{
    token:"12312312j3jijri1ojrIJIO+124+-7JROIRJ" 
}

Ответ :
{
    error_code: <error_code> 
}
